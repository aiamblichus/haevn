## Notes

- All screenshots should be taken at 1920x1080 resolution or higher
- Use consistent browser theme/colors
- Hide personal information (usernames, emails, etc.)
- Use placeholder conversations for screenshots
- Screenshots should be optimized for web (compressed but readable)
- Consider creating a dedicated test account for screenshots

## Priority

1. **High Priority**: Getting Started screenshots (installation, quick start)
2. **Medium Priority**: User Guide screenshots (syncing, archive management)
3. **Low Priority**: Platform-specific screenshots, developer diagrams

## Tools

- **Screenshots**: Chrome DevTools, Screenshot tools
- **Diagrams**: Draw.io, Excalidraw, or similar
- **Image Editing**: As needed for annotations or redactions
