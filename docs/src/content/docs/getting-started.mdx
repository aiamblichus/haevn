**HAEVN** is a Chrome extension that captures your AI conversations and stores them permanently in your browser — searchable, exportable, and entirely yours.

Every conversation you have with ChatGPT, Claude, Gemini, or any other supported platform lives only as long as that company decides to keep it. HAEVN changes that. With one click, it pulls a conversation into a local archive that never expires, never requires a subscription, and never leaves your machine.

## How it works

## What gets preserved

HAEVN captures more than plain text. For each conversation it preserves:

- All messages, in order, with timestamps
- **Images and file attachments** — both uploaded and AI-generated
- **Thinking blocks** — Claude's internal reasoning steps, when visible
- **Tool calls** — web searches, code execution, and other tool outputs
- **Code blocks** with language metadata
- Conversation branches (when you regenerate a response)
