Grok is xAI's conversational AI, integrated with X (formerly Twitter). HAEVN syncs Grok conversations via API.

## Supported URLs

- `grok.com`
- `x.ai`

## What gets synced

| Content | Supported |
|---|---|
| Text messages | ✅ |
| User-uploaded images | ✅ |
| Grok-generated images | ✅ |
| X post references | ✅ |
| Web search results | ✅ |
| Tool calls | ✅ |

## Known limitations

- **Real-time data** — Grok's real-time X data access is captured as text in the response; live post embeds are not rendered interactively in the Viewer.
- **Voice conversations** — audio is not captured; transcript only (where available).
